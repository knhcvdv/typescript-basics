# TypeScript Basics

Цей репозиторій містить базові приклади та налаштування для роботи з TypeScript.

## Опис

Проект демонструє основні можливості TypeScript, включаючи:
- Базові типи даних (string, number, boolean)
- Конфігурацію TypeScript через tsconfig.json
- Компіляцію TypeScript коду в JavaScript

## Структура проекту

```
typescript-basics/
├── src/           # Вихідний TypeScript код
├── dist/          # Скомпільований JavaScript код
├── tsconfig.json  # Конфігурація TypeScript
└── README.md      # Документація проекту
```

## Встановлення та запуск

1. Клонуйте репозиторій
2. Встановіть залежності: `npm install`
3. Скомпілюйте TypeScript: `npm run build`
4. Запустіть проект: `npm start`

## Гілки

- `main` - основна гілка зі стабільним кодом
- `develop` - гілка розробки з новими функціями
