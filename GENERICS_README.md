# Інтернет-магазин з Generic типами (коротко)

## Основне

* **Типи товарів:** `BaseProduct`, `Electronics`, `Clothing`, `Book`
* **Пошук/фільтрація:** `findProduct`, `filterByPrice`, `filterByMinPrice`, `filterByPriceRange` (generic)
* **Кошик:** `CartItem`, `addToCart`, `removeFromCart`, `updateCartQuantity`, `calculateTotal`, `getTotalQuantity` (generic)
* **Особливості:** типобезпечність, функції без класів, перевірка даних, обробка порожніх масивів
* **Демонстрація:** тестові дані, пошук, фільтрація, робота з кошиком

## Запуск

```bash
npm run build
npm run shop
```
