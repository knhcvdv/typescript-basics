# Опис інтерактивності сайту

Функціональність
-Модальні вікна (відкриття/закриття, блокування фону)
-Кнопка "вгору", плавна прокрутка, анімації при скролі
-Плавна навігація та анімація карток при кліку
-Анімації появи елементів та hover-ефекти
-Завантаження постів через Fetch API, спінер, обробка помилок, анімація, приховування кнопки
-Sticky header, адаптивність

TypeScript
Використані основні типи DOM, MouseEvent, Promise, Response, інтерфейс Post.

Запуск
Відкрити index.html або використати локальний сервер (npx serve / python -m http.server).

Структура
index.html, styles.css, src/app.ts, dist/app.js, tsconfig.json.
